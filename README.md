# Python Refresher

A comprehensive collection of Python exercises and demonstrations covering fundamental data structures and programming concepts.

## Overview

This repository contains lesson materials, guided practice exercises, and coding demonstrations for Python fundamentals, with a focus on lists, tuples, and sets.

## Files

- **Python Lesson1.py** - Introduction to basic Python concepts
- **Lesson2_Demos.py** - Demonstrations of list operations and methods
- **Lesson2_Demos_List.py** - Additional list manipulation examples
- **Lesson2_GuidedPractice.py** - Guided practice exercises for Lesson 2
- **python_set.py** - Set data structure demonstrations and operations
- **python_set_exercise.py** - Practice exercises for sets
- **python_tuple_exercise.py** - Practice exercises for tuples

## Topics Covered

### Lists
- Creating lists with mixed data types
- Positive and negative indexing
- Slicing operations
- List methods: append, remove, insert, extend, sort, count
- List concatenation and tuple conversion

### Sets
- Set creation and operations
- Set methods and properties
- Working with sets

### Tuples
- Tuple creation and access
- Immutability of tuples
- Tuple operations

## Getting Started

### Prerequisites

- Python 3.6 or higher

### Running the Code

Run any Python file using:

```bash
python filename.py
```

Example:
```bash
python Lesson2_Demos.py
python python_set_exercise.py
```

## Features

- **Mixed data type handling** - Examples demonstrate how to work with lists containing integers, floats, strings, and booleans
- **Comprehensive examples** - Each file includes print statements showing input and output
- **Progressive difficulty** - Files progress from basic demonstrations to exercises
- **Best practices** - Code includes comments explaining key operations

## Key Concepts Demonstrated

- Data type mixing in collections
- Proper sorting techniques for mixed types (using `key=str`)
- List indexing and slicing with step values
- Common list, set, and tuple operations
- Creating and converting between data structures

## Notes

- Some exercises are designed for learning purposes with multiple similar operations to reinforce concepts
- Files use descriptive variable names and comments for clarity
- Mixed-type lists use `sort(key=str)` for safe sorting across different data types
